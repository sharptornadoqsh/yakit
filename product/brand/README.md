# 睿眼品牌资源

品牌图形采用独立几何设计，由扫描边界、视域曲线与六边形中心构成，不使用上游图形。颜色、产品名和英文短名均读取 `product/renyan.json`。

执行 `yarn brand:generate` 可重新生成主标志、方形图标、浅色与深色标志、启动页静态资源、网页图标以及构建所需的多尺寸 PNG、ICO 和 ICNS。生成脚本完成后会重新读取二进制资源，检查格式与尺寸集合。

`product/brand` 保存规范资源和 Linux 多尺寸图标；`app/assets` 保存桌面窗口、托盘、关闭确认与构建图标；两套渲染器的资源目录保存界面直接引用的派生文件。派生文件不应手工编辑。
