## 三、现有服务与接口边界

账号、角色和组织页面通过 `NetWorkApi` 调用预加载桥 `axios-api`，主侧 `app/main/api/index.js` 仅校验相对路径并交给 `app/main/httpServer.js` 的 Axios 实例。实际请求地址为 `${HttpSetting.httpBaseURL}/api/`；企业模式代码会把地址设为 `https://vip.yaklang.com`。仓库中没有 `roles`、`urm`、`department` 等业务路由实现，因此这些页面不能作为可用后端证据。

项目管理通过引擎 RPC 提供 `GetProjects`、`NewProject`、`UpdateProject`、`DeleteProject` 和 `SetCurrentProject`。企业数据同步还提供 `UploadRiskToOnline`、`HTTPFlowsToOnline` 与 `GET /api/system/config`，但当前证据只覆盖上传触发和数据类型，没有多客户端共享读取、服务端项目组或受管客户端总览。
